Use the same person as in Employee (roster name). Nokia rows usually end with that employee’s name — if that name is not the same as the dropdown selection, Show Approved, Show DSM Leaves, and Mark actions will stop with an error so you can fix the copy before continuing (when the paste includes Nokia approved rows with dates).

After you change the pasted text, click Show Approved Leaves again so the table matches what will be written; Mark Approved leave in Leave tracker then applies the same rows (until you change the paste or employee). Mark Approved only creates or tags leave on Monday–Friday (Saturday/Sunday in a pasted range are skipped). Show DSM Leaves loads pending and approved rows on the Leave tracker for the selected employee across the full calendar year of the year field above (same columns as the table below; leave type uses PL/UL/SL/LL, half-day ½, and A for Nokia-approved rows, matching the month grid). Compare merges the last successful Show Approved Leaves preview with the last Show DSM Leaves load for this employee and calendar year, with leave types side by side.
